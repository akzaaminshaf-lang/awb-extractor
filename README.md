# ✈ AWB Extractor — Air Waybill Intelligence

AI-powered Air Waybill data extraction. Upload PDFs → get clean structured data instantly.

## Deploy to GitHub Pages (2 minutes)

1. Create a new GitHub repo named `awb-extractor`
2. Upload `awb-extractor.html` → rename it to `index.html`
3. Upload `README.md`
4. Go to **Settings → Pages → Source: main / root**
5. Live at `https://YOUR_USERNAME.github.io/awb-extractor`

## Usage

1. Get API key from [console.anthropic.com](https://console.anthropic.com)
2. Paste key into the app
3. Upload AWB PDFs
4. Click Extract → download CSV/JSON
